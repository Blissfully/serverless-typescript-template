# Serverless TypeScript Template

This is a template for a Serverless project using TypeScript. To get just TypeScript the dependencies, run these commands:

```sh
yarn add --exact --dev \
  @types/aws-lambda \
  @types/jest \
  @types/lodash \
  @types/node \
  @types/uuid \
  jest \
  prettier \
  ts-jest \
  ts-node \
  tslint \
  tslint-config-prettier \
  tslint-config-standard \
  typescript
```
